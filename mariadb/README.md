#### MariaDB
| 環境変数              | 意味                       | 
| --------------------- | -------------------------- | 
| MARIADB_ROOT_PASSWORD | ルートユーザーのパスワード | 
| MARIADB_DATABASE      | データベース名             | 
| MARIADB_USER          | データベースのユーザー名   | 
| MARIADB_PASSWORD      | データベースのパスワード   |
